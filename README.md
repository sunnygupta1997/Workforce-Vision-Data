# 📊 HR Insights Dashboard | Power BI Project

## 🚀 Overview
The **HR Insights Dashboard** is a dynamic and interactive Power BI project designed to transform raw employee data into meaningful business insights.  

It enables HR teams to monitor workforce trends, identify attrition patterns, and make data-driven decisions for improving organizational performance.

---

## 🎯 Objectives
- Understand employee distribution across departments  
- Analyze attrition trends and key influencing factors  
- Evaluate salary structures and patterns  
- Monitor workforce diversity and performance  

---

## 📊 Key Highlights
✔ Interactive dashboard with slicers & filters  
✔ Department-wise workforce analysis  
✔ Attrition rate & trend visualization  
✔ Salary distribution insights  
✔ Gender diversity representation  

---

## 🛠 Tech Stack
- **Power BI** (Data Visualization & Dashboarding)  
- **Microsoft Excel** (Data Cleaning & Preparation)  



---

## ⚡ How to Use
1. Download the repository  
2. Open the `.pbix` file in Power BI Desktop  
3. Interact with filters and visuals to explore insights  

---

## 💼 Business Impact
This dashboard helps organizations:
- Improve employee retention strategies  
- Identify workforce gaps  
- Make informed HR decisions  
- Optimize overall workforce planning
